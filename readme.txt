WALLET APP

Περιγραφή
Η εφαρμογή Wallet App είναι μια desktop εφαρμογή διαχείρισης προσωπικών οικονομικών, κατασκευασμένη σε Python με tkinter για το γραφικό περιβάλλον και SQLite για την αποθήκευση δεδομένων.

Βασικές λειτουργίες
- Δημιουργία λογαριασμού και σύνδεση χρήστη
- Καταχώρηση εσόδων, εξόδων, υποχρεώσεων και επιθυμιών
- Επεξεργασία και διαγραφή εγγραφών
- Ανασκόπηση εγγραφών σε πίνακες
- Φιλτράρισμα συναλλαγών ανά τύπο και κατηγορία
- Εξαγωγή δεδομένων σε αρχείο Excel
- Προβολή βασικών στατιστικών και γραφημάτων
- Εμφάνιση popup με εκκρεμείς υποχρεώσεις μετά τη σύνδεση

Οδηγίες χρήσης
1. Εκτελέστε το αρχείο wallet_app.exe.
2. Επιλέξτε "Εγγραφή" για δημιουργία νέου λογαριασμού ή "Σύνδεση" αν έχετε ήδη λογαριασμό.
3. Μετά τη σύνδεση, από το κεντρικό μενού μπορείτε να:
   - προσθέσετε νέα εγγραφή,
   - δείτε τις καταχωρημένες εγγραφές,
   - δείτε στατιστικά στοιχεία.
4. Στη "Νέα εγγραφή" επιλέγετε τύπο εγγραφής και συμπληρώνετε τα απαραίτητα πεδία.
5. Από την "Ανασκόπηση εγγραφών" μπορείτε να κάνετε επεξεργασία, διαγραφή, ανανέωση και εξαγωγή σε Excel.
6. Από τα "Στατιστικά εγγραφών" μπορείτε να δείτε σύνολα και γραφήματα.

Αποθήκευση δεδομένων
Τα δεδομένα αποθηκεύονται τοπικά σε βάση SQLite στο αρχείο wallet_app.db.

Απαιτήσεις
Η εφαρμογή έχει σχεδιαστεί για περιβάλλον Windows.

Γνωστοί περιορισμοί
- Τα δεδομένα αποθηκεύονται μόνο τοπικά.
- Δεν υποστηρίζεται συγχρονισμός μεταξύ διαφορετικών συσκευών.
- Οι κωδικοί αποθηκεύονται χωρίς προηγμένη κρυπτογράφηση.
- Η εφαρμογή δεν έχει δοκιμαστεί εκτενώς σε όλα τα περιβάλλοντα Windows.
